- **Delivered** plans get moved here with a "Status: delivered" banner pointing to the current reference docs
- **Rejected** plans get moved here with a "Status: rejected" banner explaining why we didn't pursue them
- **Superseded** plans get moved here with a "Status: superseded" banner pointing to whatever replaced them
